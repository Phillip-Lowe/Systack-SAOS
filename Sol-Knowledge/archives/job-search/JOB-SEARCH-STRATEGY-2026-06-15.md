## UPWORK: AUTHENTICATION ISSUE & SOLUTION

### The Problem
You tried Upwork before but "got stuck in authentication." Common issues:

1. **ID Verification** — Requires government-issued photo ID + proof of address
2. **Camera Access** — Need webcam for live photo verification
3. **Location Verification** — For US-only jobs, may need additional verification
4. **"Verifying You Are Human"** — Captcha/security checks

### Requirements
| Item | What You Need |
|------|---------------|
| **Government ID** | Driver's license, passport, state ID (must be valid, not expired) |
| **Proof of Address** | Bank statement, utility bill, lease (within last 3 months, shows your name) |
| **Webcam/Camera** | For live selfie during verification |
| **Good Lighting** | Clear photo of ID and your face |

### Step-by-Step Fix
1. **Gather documents:**
   - Valid driver's license or passport
   - Recent bank statement or utility bill (matches ID address)
   
2. **Use a proper device:**
   - Desktop/laptop with webcam OR
   - Phone with clear camera
   - **Not:** Blurry photos, screenshots, expired IDs

3. **Start verification:**
   - Go to upwork.com → Settings → Identity Verification
   - Follow prompts: upload ID → take selfie → upload proof of address

4. **If failed before:**
   - Check email for specific failure reason
   - Common: blurry photo, expired ID, name mismatch, address mismatch
   - Fix issue → re-submit (usually allowed)

5. **If stuck/no option to resubmit:**
   - Contact Upwork Support directly
   - Email: support@upwork.com
   - Include: username, issue description, ready to provide docs

### Alternative: Create New Account?
- **Not recommended** — Upwork links accounts by identity
- Better to resolve existing account
