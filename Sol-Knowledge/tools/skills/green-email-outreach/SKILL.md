# Green Email Outreach

Systack prospecting email outreach for Green's business. Personalizes emails per lead, tracks sends, manages drip follow-ups, and logs everything to CSV.

## Pipeline

1. **Load leads** — from Green lead scraper output or manual CSV
2. **Personalize** — Green/Systack templates with lead variables
3. **Send** — via Resend API with rate limiting
4. **Track** — log every send to CSV with timestamp and status
5. **Drip** — automated follow-ups at day 3 and day 7
6. **Report** — campaign stats and lead engagement summary

## Requirements

- Node.js 18+
- `RESEND_API_KEY` environment variable
- Verified sending domain (configured in Resend)
- Lead CSV with columns: `first_name`, `email`, `company`, `domain`

## Quick Start

```bash
# Set API key
export RESEND_API_KEY=re_xxxxxxxx

# Send campaign
node scripts/outreach.js --source leads.csv --template templates/systack-intro.txt --from "Green <green@systack.net>"

# Dry run (preview only)
node scripts/outreach.js --source leads.csv --template templates/systack-intro.txt --dry-run

# Drip follow-up (day 3)
node scripts/outreach.js --drip --days 3 --template templates/systack-followup.txt

# Generate campaign report
node scripts/report.js --log send-log.csv
```

## Templates

Templates live in `templates/` and use `{variable}` syntax.

### Available Variables

- `{first_name}`, `{last_name}`, `{email}`
- `{company}`, `{domain}`
- `{sender_name}`, `{sender_title}`
- `{pain_point}` — auto-generated from company website

### Green/Systack Templates

- `templates/systack-intro.txt` — Initial outreach
- `templates/systack-followup.txt` — Day 3 follow-up
- `templates/systack-final.txt` — Day 7 final touch

### Template Example

```
Subject: {company} — quick question about your online presence

Hi {first_name},

I came across {company} and noticed {pain_point}. We help businesses 
like yours build automated systems that save hours every week.

Would it make sense to chat for 10 minutes this week?

Best,
{sender_name}
Systack Automation
```

## Configuration

Edit `scripts/config.json`:
- `maxPerDay`: 25 (daily limit per domain)
- `delayBetweenMs`: 3000 (3 seconds between emails)
- `dripDays`: [3, 7] (follow-up schedule)
- `suppressionFile`: Path to unsubscribe list
- `trackingFile`: Path to send log CSV

## Output

### Send Log CSV
```csv
timestamp,email,company,status,template,message_id
2026-06-05T06:00:00Z,contact@greenlandscaping.com,Green Landscaping,sent,systack-intro,re_abc123
```

### Campaign Report
Generated by `scripts/report.js`:
- Total sent
- Bounce rate
- Open rate (if tracking enabled)
- Reply rate (manual tracking)
- Day-by-day breakdown

## Compliance

- CAN-SPAM: physical address in footer
- Unsubscribe link in every email
- Suppression list checked before every send
- Rate limiting: max 25/day per domain
