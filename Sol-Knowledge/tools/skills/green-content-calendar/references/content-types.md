# Content Type Guidelines

## Blog Post
- **Length:** 800-1200 words
- **Format:** Long-form, educational
- **SEO:** Include target keywords, meta description
- **CTA:** Book demo, download guide, subscribe

## Social Post
- **Length:** 50-150 words
- **Format:** Short, engaging, visual
- **Platform:** LinkedIn, Twitter, Instagram
- **CTA:** Visit website, comment, share

## Email Newsletter
- **Length:** 300-600 words
- **Format:** Personal, conversational
- **Segments:** Prospects, customers, partners
- **CTA:** Book call, read blog, refer friend

## Video
- **Length:** 2-5 minutes
- **Format:** Tutorial, testimonial, demo
- **Platform:** YouTube, LinkedIn, website
- **CTA:** Subscribe, book demo, download
