## USAGE NOTES

**For Image Generation:**
- All prompts specify the exact brand hex colors
- Backgrounds default to navy (#001a2d) or white (#ffffff) for consistency
- Accent colors are used sparingly and purposefully
- Green (#22c55e) reserved for success/validation states
- Red (#ef4444) reserved for warnings/Pessi/alert contexts
- Purple (#8b5cf6) reserved for creative/Geni contexts

**Recommended Priority Order:**
1. Business Fleet Hero (your most popular tier)
2. The Growth Path (shows all tiers, landing page hero)
3. Fleet Command / SOL (brand identity)
4. Tier Card Set (pricing page)
5. Order Agent feature image
6. SAOS Security Shield (trust/privacy differentiator)
