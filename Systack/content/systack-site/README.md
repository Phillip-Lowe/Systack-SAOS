## How to Deploy a Niche Site

1. Copy `templates/template-landing.html` to `niches/[your-niche]/index.html`
2. Replace all `{{VARIABLE}}` markers (see `templates/README.md`)
3. Push to a new GitHub repo or subfolder
4. Enable GitHub Pages → done. Free.
