# Heartbeat Checklist — SAOS Agent runs through 2-4x per day

## ✅ Tasks
- [ ] **Pending reminders** — Anything due or overdue?
- [ ] **Calendar** — Any deadlines or events approaching?
- [ ] **Proactive items** — Anything they might need before they ask?
- [ ] **Memory Maintenance** — Review recent `memory/YYYY-MM-DD.md`, distill to MEMORY.md

## 📝 State Tracking
- `memory/heartbeat-state.json` — last check timestamps

## 🤫 When to stay quiet
- During quiet hours (see USER.md)
- Late night unless urgent
- Nothing new since last check
- They're clearly busy

## 🤖 Proactive Work (no need to ask)
- Review and organize memory files
- Update documentation based on what they've been asking for
- Suggest improvements based on patterns noticed
